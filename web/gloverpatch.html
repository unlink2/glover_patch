Controls

Controller 1

  • Start + B: Level Select
  • Start + A: Start Timer
  • Start + C-Left/C-Right: Store/Load Position
  • Start + C-Up/C-Down: Store/Restore Actors
  • Start + Z: Toggle input viewer

Controller 2

  • Start: Open/Close menu
  • Navigate menu with D-Pad Up/Down and A/B
  • Values can be incremented/decremented with D-Pad Left/Right
  • C-Up: Toggle Frame Advance
  • C-Down: Advance a frame

Memory Viewer (Controller 2)

  • Navigate with D-Pad Left/Right
  • D-Pad Up/Down: Increment/Decrement value
  • B: Add an address to watch
  • Z: Input a value
  • A: Follows a pointer
