Glover Patch Tool

Glover Practice ROM patching tool usage:

  • Select a ROM file in z64 format
  • The tool will patch the ROM (This might take a few seconds)

Is this legal?

Yes! The ROM file never leaves your computer. Patching is entirely done locally and I do not distribute any Nintendo 64 ROM files. It is the user's responsibility to provide a legit ROM.
